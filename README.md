## Image Converter App

This is a simple application for converting images between different formats (JPG, PNG, WebP, GIF).

## Getting Started

You can run it locally by cloning the repository and running the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
